# LEVEL FIT - Сайт подписок на онлайн тренировки и питание

Премиум сайт для продажи подписок на онлайн тренировки и программы питания.

## Структура проекта

```
levelfit/
├── index.html          # Главная страница
├── styles.css          # Стили сайта
├── script.js           # JavaScript функциональность
├── images/             # Папка для изображений
│   ├── hero-1.jpg     # Фото для hero секции (5 изображений)
│   ├── hero-2.jpg
│   ├── hero-3.jpg
│   ├── hero-4.jpg
│   ├── hero-5.jpg
│   ├── gallery-1.jpg  # Фото для галереи (5 изображений)
│   ├── gallery-2.jpg
│   ├── gallery-3.jpg
│   ├── gallery-4.jpg
│   └── gallery-5.jpg
└── README.md           # Этот файл
```

## Установка изображений

Для корректной работы сайта необходимо добавить изображения в папку `images/`:

1. **Hero изображения** (hero-1.jpg до hero-5.jpg):
   - Используйте предоставленные фотографии тренировок
   - Рекомендуемый размер: 1920x1080px или больше
   - Формат: JPG, оптимизированные для веба

2. **Галерея** (gallery-1.jpg до gallery-5.jpg):
   - Можно использовать те же фотографии или другие
   - Рекомендуемый размер: 800x600px или больше

3. **Логотип**:
   - Логотип LEVEL FIT встроен в CSS как текст с золотым цветом
   - При необходимости можно заменить на изображение

## Варианты подписок

Сайт предлагает 4 варианта подписок:

1. **Тренировки** - 1990₽/мес
   - Доступ к библиотеке тренировок
   - Программы для всех уровней
   - Видео-инструкции
   - Трекинг прогресса

2. **Питание** - 1490₽/мес
   - Персональный план питания
   - Рецепты и меню
   - Калькулятор калорий
   - Трекинг питания

3. **Тренировки + Питание** - 2990₽/мес (Популярный)
   - Все из плана "Тренировки"
   - Все из плана "Питание"
   - Синхронизация программ
   - Комплексный подход

4. **Всё включено** - 4990₽/мес (Премиум)
   - Все из плана "Тренировки + Питание"
   - Персональная связь с тренером
   - Еженедельные консультации
   - Корректировка программы
   - Приоритетная поддержка
   - Эксклюзивные материалы

## Запуск сайта

1. Добавьте изображения в папку `images/`
2. Откройте `index.html` в браузере
3. Или используйте локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (если установлен http-server)
   npx http-server
   ```

## Кастомизация

- **Цвета**: Измените переменные в `styles.css` (секция `:root`)
- **Цены**: Обновите значения в `script.js` (объект `plans`)
- **Тексты**: Редактируйте содержимое в `index.html`
